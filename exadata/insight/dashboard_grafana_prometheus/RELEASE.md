## 25.2 - 2025-12-09
This release introduces several key changes

### Changes
* Added dataSource dashboard variable to all dashboards.  This allows the user to select a prometheus datasource
* Added new dashboards
-  CellDiskAggr.json - shows cell disk metrics aggregated per cell
-  DiskIO.json - shows various IO metrics that can cause disk IO
-  IOProfile.json - shows both cell disk and IO statistics metrics in a single dashboard
-  IOStatisticsAggr.json - shows IO statistics aggregated per cell
