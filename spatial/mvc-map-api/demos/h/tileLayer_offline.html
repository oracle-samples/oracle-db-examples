Maps API Demos
This demo displays a pre-generated tile layer, stored on the local machine. Displaying the layer needs neither map viz server, nor network connection. A use case scenario is: a user has a web server running on a portable machine with the pre-geneted map viz tiles deployed into the web server. The portable machine can access the tiles locally so the user may bring the machine to the field, and is able to access the maps tiles. The map viz server has utilities to generate and export the tiles.
