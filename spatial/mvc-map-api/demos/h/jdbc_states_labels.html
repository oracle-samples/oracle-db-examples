Maps API Demos
This demo shows a thematic map. On top of a background map from Oracle Maps Cloud Service World Map, it has two JDBC vector layers, one for population density of each state, the other, its total population. It has a map control, the vector layer control, which allows you to turn a vector layer on/off. A custom color ramp is used to color-code the population density. For the total population layer, a circle with a text label is used, and the circle size is determined by the state's total population. An InfoWindow shows a state's info upon a mouse click.
